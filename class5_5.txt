Question 1: Which part of the computer is known as its "brain"?
1) Monitor
2) Keyboard
3) CPU
Answer: Option3: CPU
Question 2: What is the process of starting a computer called?
1) Shutting down
2) Booting
3) Logging off
Answer: Option2: Booting
Question 3: Which memory is temporary and loses data when the power is turned off?
1) ROM
2) RAM
3) Hard Disk
Answer: Option2: RAM
Question 4: What is the key combination to copy selected text?
1) Ctrl + V

Question 6: What kind of software is Microsoft Word?
1) System Software
