1. What is the name of the largest planet in our solar system?
A) Earth
B) Saturn
C) Jupiter
D) Mars
Answer: Option 3: Jupiter
2. Which part of a computer do we call the brain?
A) Hard drive
B) CPU (Central Processing Unit)
C) Memory
D) Power supply
Answer: Option 1: CPU (Central Processing Unit)
3. What is the highest level of organization in a school?
A) Grade
B) Class
C) School
D) Region
Answer: Option 2: Grade
4. Why do we use batteries in our toys and gadgets?
A) Because they make them lighter
B) Because they help us store energy
C) Because they give us power to play
D) Because they can be used as a toy too!
Answer: Option 1: Because they make them lighter
5. How do we tell computers which programs to run when we turn on the computer?
A) We type in the program name
B) We type in the command line
C) We press Start button
D) We type a letter 'Y' and then our favourite program name!
Answer: Option 1: Because we type in the program name
6. What is an example of computer science that we can use to help people?

C) Creating games
D) Making robots walk!

A) To type documents
B) To play video games
C) To help people communicate with others using computers
D) All of above, except one because we can't use it to help people without a key pressed!
